# Hocus Pocus - Setup

## Environment Variables

Required:

- `ADMIN_SECRET` - secret for admin login
- `SUPABASE_URL` - project URL (e.g. `https://xyzcompany.supabase.co`)
- `SUPABASE_SERVICE_ROLE_KEY` - service role key (server-side only)
- `BREVO_API_KEY` - Brevo transactional email API key

Optional:

- `EMAIL_FROM` - sender email (default `content@frontira.io`)
- `EMAIL_FROM_NAME` - sender name (default `Hocus Pocus`)
- `APIFY_API_TOKEN` - enables LinkedIn profile name enrichment
- `SUPABASE_APPLICATIONS_TABLE` (default `applications`)
- `SUPABASE_MEMBERS_TABLE` (default `members`)
- `SUPABASE_INVITES_TABLE` (default `invites`)
- `SUPABASE_EVENTS_TABLE` (default `events`)

## SQL Schema

Run this in Supabase SQL editor:

```sql
create table if not exists applications (
  id bigint generated by default as identity primary key,
  email text not null,
  linkedin text not null,
  status text not null default 'pending',
  "createdAt" timestamptz not null default now(),
  source text not null default 'public',
  "inviteToken" text,
  "approvedAt" timestamptz,
  "rejectedAt" timestamptz,
  "memberId" bigint,
  "memberToken" text
);

create table if not exists members (
  id bigint generated by default as identity primary key,
  name text,
  email text not null,
  linkedin text not null,
  "accessToken" text not null unique,
  "createdAt" timestamptz not null default now(),
  "inviterMemberId" bigint,
  "applicationId" bigint
);

create table if not exists invites (
  id bigint generated by default as identity primary key,
  token text not null unique,
  "memberId" bigint,
  "createdAt" timestamptz not null default now(),
  "expiresAt" timestamptz not null,
  "usedAt" timestamptz,
  "claimedByMemberId" bigint,
  "recipientEmail" text,
  method text default 'link',
  "senderPersona" text
);

create table if not exists guestlist (
  id bigint generated by default as identity primary key,
  email text not null,
  linkedin text,
  "senderPersona" text default 'joanna',
  status text not null default 'draft',
  "createdAt" timestamptz not null default now(),
  "sentAt" timestamptz,
  "inviteId" bigint
);

create index if not exists applications_email_idx on applications (email);
create index if not exists members_access_token_idx on members ("accessToken");
create index if not exists invites_member_id_idx on invites ("memberId");
create index if not exists invites_token_idx on invites (token);
create index if not exists guestlist_status_idx on guestlist (status);

create table if not exists events (
  id bigint generated by default as identity primary key,
  type text not null,
  actor text,
  target text,
  data jsonb,
  "createdAt" timestamptz not null default now()
);

create index if not exists events_type_idx on events (type);
create index if not exists events_target_idx on events (target);
create index if not exists events_created_idx on events ("createdAt" desc);
```

If migrating an existing database, run:

```sql
ALTER TABLE members ADD COLUMN IF NOT EXISTS name text;
ALTER TABLE invites ADD COLUMN IF NOT EXISTS "recipientEmail" text;
ALTER TABLE invites ADD COLUMN IF NOT EXISTS method text DEFAULT 'link';
ALTER TABLE invites ADD COLUMN IF NOT EXISTS "senderPersona" text;
ALTER TABLE invites ALTER COLUMN "memberId" DROP NOT NULL;

-- Guest list table (new)
create table if not exists guestlist (
  id bigint generated by default as identity primary key,
  email text not null,
  linkedin text,
  "senderPersona" text default 'joanna',
  status text not null default 'draft',
  "createdAt" timestamptz not null default now(),
  "sentAt" timestamptz,
  "inviteId" bigint
);
create index if not exists guestlist_status_idx on guestlist (status);

-- Event log table (new)
create table if not exists events (
  id bigint generated by default as identity primary key,
  type text not null,
  actor text,
  target text,
  data jsonb,
  "createdAt" timestamptz not null default now()
);
create index if not exists events_type_idx on events (type);
create index if not exists events_target_idx on events (target);
create index if not exists events_created_idx on events ("createdAt" desc);
```

## URLs

- Public landing: `/`
- Private event page: `/inside.html?token=<memberToken>`
- Invite claim: `/inside.html?invite=<inviteToken>`
- Admin: `/admin-login.html` -> `/admin.html`

## Notes

- Guest invite links expire after 24 hours, max 2 per member.
- Admin proactive invites expire after 7 days, no quota limit.
- Invite emails show "Full Name / email invited you" using LinkedIn-enriched names.
- If Supabase variables are missing, APIs fall back to in-memory storage (local testing only).
